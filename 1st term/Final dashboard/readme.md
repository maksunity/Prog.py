Проект заключен в отслеживании жизненных показателей пациента. 
Сделана авторизация(пароль хэшируется), регистрация(одно имя пользователя - один человек, один телефон - один человек), валидация данных (номер это 11 цифр, маска для почта, ограничения на показания).
Личный кабинет для занесения роста и веса, также отображение личных данных.
Бэкенд: Flask,графики реализованы как в grafana так и в plotly, база данных postgres.
